# TikTokParser

```bash
 Use --parser, -p to choose parse, available parsers:
  snaptik, ssstik, ttdownloader, musicaldown, savefrom, tiktokfull, api-wrapper, api-snaptik
 Use word random to use random parser
 Use --headless, -h to see the parsing process
 Use --url, -u to parse url
 Use --short-url, -s to shorten the link
 Use --disable-gpu, -d to don't use gpu
 Use --output-file, -o to set file in which the link will be saved
 Use --set-browser-binary-path, -set-binary set browser binary path
 Use --browser, -b to choose browser, available browsers: firefox, chromium

 DEFAULT: parser random, headless true, shorting url false, browser firefox, gpu disabled
 ```
